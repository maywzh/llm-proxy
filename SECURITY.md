# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | yes       |
| < 1.0   | no        |

## Reporting a Vulnerability

Please do NOT report security vulnerabilities through public GitHub issues.

Instead, open a [GitHub Security Advisory](https://github.com/maywzh/llm-proxy/security/advisories/new).

You should receive a response within 48 hours. Please include:

- Type of issue (buffer overflow, injection, XSS, etc.)
- Full path(s) of affected source file(s)
- Location (tag/branch/commit or direct URL)
- Steps to reproduce
- Proof-of-concept or exploit code if possible
- Impact assessment
